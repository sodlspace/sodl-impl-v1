SODL is an open project. We welcome feedback, questions, and contributions.

## GitHub

The best place to report bugs, request features, or contribute:

[github.com/sodl-lang/sodl](https://github.com/sodlspace/sodl-impl-v1)

- Open an issue for bugs or feature requests
- Submit a pull request for improvements
- Star the repo to follow progress

## Email

## Status

SODL is in active development (v1.0.0). The language specification and tooling are evolving. Breaking changes may occur before v1.1.0.

Follow the [GitHub repository](https://github.com/sodlspace/sodl-impl-v1) for release announcements.
